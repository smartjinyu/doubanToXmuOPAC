doubanToXmuOPAC
===============
##功能说明：##
在豆瓣查看一本书/刊的详细信息页面中的右侧菜单栏，增加“在哪借这本书”列表，显示该书在厦大图书馆具体馆藏信息，点击后直接链接到厦大图书馆的书目查询系统。
具体使用方式和效果请看http://news.xmulib.org/2008/12/25/post-30/ 

	1. 如果图书馆有这本书，则直接在豆瓣页面中加载这本书的详细信息 
	2. 如果没有这本书，但有同名图书的，则返回同名书列表和在厦大图书馆的链接
